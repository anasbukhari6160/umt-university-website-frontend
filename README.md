# UMT University Website

This project is a static university website developed as part of an academic assignment. It is designed using basic web technologies and focuses on clean structure, responsiveness, and user-friendly layout.

##  Technologies Used
- HTML5
- CSS3
- Bootstrap

##  Features
- Home page with introduction
- Admissions page
- Faculty information page
- Programs page
- Contact section
- Fully responsive design

## Project Structure
- index.html
- admission.html
- faculty.html
- program.html
- style.css
- images/

## Purpose
This project was created for learning and demonstration purposes. It follows basic web development practices and is suitable for viva explanation.

## Deployment
The website is deployed using Netlify for live preview.
